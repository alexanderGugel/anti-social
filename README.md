# anti-social

A quick-and-dirty command line utility that unfollows everyone on GitHub.

## Install

Via [`npm`](https://www.npmjs.org/) do:

```
  npm i anti-social -g
```

## Usage

```
  anti-social
```

Enter your login credentials and wait.
